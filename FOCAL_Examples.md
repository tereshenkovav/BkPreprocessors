# Примеры обработки исходных кодов Фокала препроцессором

## Обработка комментариев, пустых строк, включаемых файлов и сокращение ключевых слов

Исходный файл:
```
# Демонстрация использования процедур клавиатуры

// Отключение прерываний клавиатуры
1.10 DO 80
1.20 TYPE "PRESS ANY KEY OF ENTER TO EXIT" !
// Ожидание нажатой клавиши
1.30 SET K = FSBR(81,0)
1.40 IF [K-10] 1.50,1.70
1.50 TYPE K,!
1.60 GOTO 1.30
1.70 QUIT

INCLUDE proc_keyboard.foc
```

Включаемый файл proc_keyboard.foc
```
# Модуль работы с клавиатурой

// Запрет на прерывания
80.10 XECUTE FX(-1,177660,64)

// Ожидание клавиши и возврат код после вызова функции
81.10 IF [FX(0,177660,128)] 81.10,81.10,81.20
81.20 SET &=FX(1,177662)
```

Обработанный файл:
```
1.10 D 80
1.20 T "PRESS ANY KEY OF ENTER TO EXIT" !
1.30 S K = FSBR(81,0)
1.40 I [K-10] 1.50,1.70
1.50 T K,!
1.60 G 1.30
1.70 Q
80.10 X FX(-1,177660,64)
81.10 I [FX(0,177660,128)] 81.10,81.10,81.20
81.20 S &=FX(1,177662)
```
